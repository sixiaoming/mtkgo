## 代码结构:
```
  mktgo
    ├── bin                 ---- 启停脚本
    │   ├── dev
    │   ├── test
    │   └── prop
	├── conf                ---- 全局配置文件
    │   ├── dev
    │   ├── test
    │   └── prop	
    ├── src                 ---- 业务代码
    │   ├── aop          	---- 日志切面
    │   ├── config          ---- 各种依赖组件集中化配置    
    │   ├── pojo          	---- 各种实体
    │   ├── service     	---- 业务层
    │   ├── schedle         ---- 调度任务
    │   ├── support         ---- 各种支持工具类
    │   └── web				---- 控制路由层   
  	└── pom    
```
